# Credit Risk Probability Model for BNPL

This project implements an end-to-end machine learning pipeline for predicting credit risk using alternative data. Built as part of the 10 Academy AI Mastery Program, it supports Buy-Now-Pay-Later services for Bati Bank in partnership with an eCommerce provider.

## 📌 Project Goals

- Define proxy risk labels from transaction behavior
- Perform EDA and feature engineering
- Train interpretable and high-performance models
- Serve model using FastAPI
- Automate testing and deployment using CI/CD
